# chatbot_curriculo_ai
ChatBot que responde como se fosse eu, sobre meu próprio currículo

Resumo:
Chatbot baseado em IA que responde perguntas sobre meu currículo utilizando dados estruturados do Google Sheets e automação via n8n.

Problema a resolver:
Automatiza respostas personalizadas para recrutadores, evitando respostas genéricas e melhorando a apresentação profissional.

Tecnologias usadas:
- Python / JavaScript
- OpenAI API
- n8n
- Google Sheets

Aprendizados:
- Integração de APIs
- Engenharia de prompt
- Automação de workflows
- Estruturação de dados para IA

Melhorias futuras:
- Deploy em web app
- Integração com LinkedIn
- Interface gráfica
